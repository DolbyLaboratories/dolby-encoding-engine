Dolby Encoding Engine plugin example for scaling.

Build tools: 
- Visual Studio 2013
- gcc 4.8.5

Build instructions, Windows:
1. Build using make\scaling_example\windows_amd64_msvs\scaling_example_2013.sln
2. Copy dll library with plugin to DEE installation folder

Build instructions, Linux:
1. Build using make/scaling_example/linux_amd64_gnu/Makefile
    - make scaling_example_release.so
2. Copy so library with plugin to DEE installation folder
